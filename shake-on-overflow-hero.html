Why the page looks empty
This demo loads ShakeOnOverflowHero.js as an ES module. Browsers block that when you open the file directly (file://).

Fix: run a tiny local server from this project folder, then open the link below.

chmod +x preview-hero.sh
./preview-hero.sh

Or: python3 -m http.server 8765

Then open http://127.0.0.1:8765/shake-on-overflow-hero.html (port must match).

In VS Code / Cursor you can also use “Live Preview” or the Live Server extension on this HTML file.
